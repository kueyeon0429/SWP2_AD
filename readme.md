# 가위바위보 게임
----------------
### SWP2 AD project
### 20203060민경서 20203065박규연
---------------
## 소개
> 추억의 짱깸뽀 게임을 PyQt5로 구현
---------------
## 게임 설명
> 1. START버튼 클릭: 300원(디폴트)로 게임 시작
> 2. 가위, 바위, 보 버튼 클릭
> 3. 승패무 결정
>    * 승: 랜덤 금액 지급
>    * 패: 0원 지급
>    * 무: 100원 지급
> 4. 잔액이 0원 or 1000원일 경우 결과 확인 버튼 활성화
>    * 0원: 거지가 되었습니다.
>    * 1000원: 부자가 되었습니다.
> 5. CLOSE버튼 클릭: 게임 종료
